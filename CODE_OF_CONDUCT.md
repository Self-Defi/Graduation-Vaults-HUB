# Code of Conduct

## Our Standard
Graduation Vaults is built under the Self-Defi principle: **sovereignty first** and **zero custody**.
We expect everyone contributing or participating in this project to be respectful, direct, and constructive.

## We Will
- Be respectful and professional in language and tone.
- Provide clear, actionable feedback.
- Assume good intent, verify with facts.
- Keep discussions focused on improving the project.

## We Will Not
- Harass, insult, or demean others.
- Post threats, doxxing, or personal attacks.
- Promote scams, phishing, or deceptive behavior.
- Encourage unsafe custody behavior.

## Scope
This applies to all project spaces: GitHub issues, discussions, pull requests, docs, and any public collaboration channel.

## Reporting
If you see behavior that violates this Code of Conduct, report it privately:
- Email: **support@graduationvaults.com**

## Enforcement
Violations may result in warnings, comment removal, issue/PR closure, or contributor bans depending on severity.
